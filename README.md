## 주요 기능

- **소셜 로그인**:
  - 사용자 인증 정보를 통해 로그인합니다.
  - Google과 Apple 로그인 가능.

- **동아리 일정관리**:
  - 캘린더 UI를 통해 동아리 관련 일정 추가&삭제가 가능합니다.

- **알림 전송**:
  - FCM을 통해 알림을 발송할 수 있습니다.
  - 공지사항 및 댓글이 달렸을 경우 알림이 발송됩니다.

- **글 검색**:
  - 키워드를 통해 특정 글 검색이 가능합니다.

- **사진 업로드**:
  - 최대 10장의 사진을 게시글과 함께 업로드 할 수 있습니다.

- **동아리 SNS 이동**:
  - 동아리의 인스타 및 네이버 카페로 이동이 가능합니다.

- **익명 프로필**:
  - 모든 사용자는 철저하게 익명으로 활동하며 원하는 닉네임과 사진으로 언제든 변경할 수 있습니다.

- **내가 작성한 글 및 댓글 찾기**:
  - 내가 작성한 글과 댓글을 찾을 수 있습니다.
